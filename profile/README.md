# Swipe 💳
> 👨‍💻 Project   

### 🐲 개요
-------------
```
청년들이 보다 효율적이고 안전한 금융 서비스를 선택할 수 있도록 돕기 위한 것입니다.

우리는 카드, 보험, 적금에 관한 정보를 종합적으로 제공하여 사용자들이 개인적인 필요에 맞게 최적의 금융 결정을 할 수 있도록 지원합니다.
```
